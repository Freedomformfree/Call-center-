API Documentation

Complete API reference for agentic functions integration

API Overview

The Agentic Functions API provides programmatic access to all automation capabilities. Use these endpoints to integrate with your applications, create custom workflows, and manage function executions.

Base URL: https://api.agentic-functions.com/v1
Authentication: Bearer Token
Content-Type: application/json
Rate Limits: 1000 requests/hour
Response Format: JSON
Status Codes: Standard HTTP codes
Function Execution
POST /api/functions/execute

Execute any available agentic function with custom parameters.

Request Body:
{ "function_name": "email_sender", "context": { "to": ["user@example.com"], "subject": "Test Email", "content": "Hello from AI!" }, "client_phone": "+1234567890" }
Response:
{ "success": true, "data": { "message_id": "msg_123456", "recipients": ["user@example.com"], "status": "sent" }, "error": null, "metadata": { "execution_time": 1.23, "function_version": "1.0.0" }, "timestamp": "2023-12-01T10:30:00Z" }
Connection Management
POST /api/connections

Create connections between functions for automated workflows.

Request Body:
{ "source_function": "data_analyzer", "target_function": "email_sender", "connection_type": "sequential", "conditions": { "require_success": true }, "mapping": { "analysis_result": "email_content" } }
GET /api/connections

Retrieve all function connections.

DELETE /api/connections/{connection_id}

Delete a specific connection.

Client Management
POST /api/clients

Connect a client phone to agentic functions.

Request Body:
{ "phone_number": "+1234567890", "client_name": "John Doe", "functions": ["email_sender", "data_analyzer"], "enable_gemini": true, "auto_trigger": true, "trigger_keywords": ["automation", "data"] }
GET /api/clients

Retrieve all connected clients.

DELETE /api/clients/{phone_number}

Disconnect a client phone.

Workflow Management
POST /api/workflows

Create a new automated workflow.

Request Body:
{ "name": "Daily Report Generation", "description": "Automated daily data analysis and reporting", "steps": [ { "function": "data_analyzer", "context": {"data_source": "daily_metrics"}, "continue_on_error": false }, { "function": "email_sender", "context": {"template": "daily_report"}, "continue_on_error": true } ] }
POST /api/workflows/{workflow_id}/execute

Execute a specific workflow.

GET /api/workflows

Retrieve all workflows.

Available Functions
GET /api/functions

Get list of all available functions with their parameters and descriptions.

Response Example:
{ "email_sender": { "name": "Email Sender", "description": "Send emails via SendGrid with templates", "category": "communication", "parameters": { "to": "array of email addresses", "subject": "email subject", "content": "email content", "template": "optional template name" } }, "data_analyzer": { "name": "Data Analyzer", "description": "Analyze datasets with statistical insights", "category": "data", "parameters": { "data_source": "data source identifier", "analysis_type": "type of analysis to perform" } } }
Error Handling

The API uses standard HTTP status codes and returns detailed error information:

Error Response Format:
{ "success": false, "error": "Function 'invalid_function' not found", "error_code": "FUNCTION_NOT_FOUND", "details": { "available_functions": ["email_sender", "data_analyzer", "..."] }, "timestamp": "2023-12-01T10:30:00Z" }
Common Error Codes:
  • FUNCTION_NOT_FOUND - Requested function does not exist
  • INVALID_PARAMETERS - Function parameters are invalid
  • CLIENT_NOT_FOUND - Client phone number not connected
  • RATE_LIMIT_EXCEEDED - API rate limit exceeded
  • AUTHENTICATION_FAILED - Invalid or missing authentication

import requests # Execute a function response = requests.post( 'https://api.agentic-functions.com/v1/api/functions/execute', headers={ 'Authorization': 'Bearer YOUR_API_TOKEN', 'Content-Type': 'application/json' }, json={ 'function_name': 'email_sender', 'context': { 'to': ['user@example.com'], 'subject': 'Test Email', 'content': 'Hello from Python!' } } ) result = response.json() print(f"Success: {result['success']}")
// Execute a function const response = await fetch('https://api.agentic-functions.com/v1/api/functions/execute', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_TOKEN', 'Content-Type': 'application/json' }, body: JSON.stringify({ function_name: 'email_sender', context: { to: ['user@example.com'], subject: 'Test Email', content: 'Hello from JavaScript!' } }) }); const result = await response.json(); console.log(`Success: ${result.success}`);
